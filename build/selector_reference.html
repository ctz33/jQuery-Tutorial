jQuery Selectors Reference

Basic Selectors

Selector Description Example Selects
$("element") Selects all elements with the specific tag name. $("p") All <p> elements in the document
$("#id") Selects a single element with the specific id. $("#test-id") The single element that has an id "test-id"
$(".class") Selects all elements with the specific class. $(".test-class") All elements that have a class "test-class"
$("*") Selects all elements. $("*") All elements
$("selector 1, selector 2, selector N") Selects elements that match any one of the selectors. $(".test-class, #test-id") Elements that have a class "test-class" or have an id "test-id"
$(":even") Selects elements with even indices (index starts with 0). $("li:even") Even <li> elements
$(":odd") Selects elements with odd indices (index starts with 0). $("li:odd") Odd <li> elements
$(":root") Selects the root element of the document. The root element of HTML is always the <html> element. $(":root") The root element of the document.
$(":first") Selects the first element. $("li:first") The first <li> element
$(":last") Selects the last element $("li:last") The last <li> element
$(":eq(index)") Selects the element with the specific index (index starts with 0). $("p:eq(0)") The first <p> element
$(":gt(index)") Selects all elements at indices that are greater than the specific index (index starts with 0). $("p:gt(4)") All <p> elements that have indices greater than 4
$(":lt(index)") Selects all elements at indices that are less than the specific index (index starts with 0). $("p:lt(4)") All <p> elements that have indices less than 4
$(":not(selector)") Selects all elements that do not match the specific selector. $("p:not(.test-class)") All <p> elements that do not have a class "test-class"
$(":header") Selects all header elements, i.e. <h1>, <h2>, <h3>, etc. $(":header") All header elements
$(":animated") Selects all elements that are currently animated. $(".changed:animated") All elements with class "changed" that are currently animated
$(":lang(language)") Selects all elements with the language attribute starting with the specific value. $("p:lang(en)") All <p> elements that have a language attribute whose value starts with "en"

Attribute Selectors

Selector Description Example Selects
$("[attribute]") Selects all elements that have the specific attribute. $("[type]") All elements that have a type attribute
$("[attribute=value]") Selects all elements that have the specific attribute with the specific value. $("[type='text']") All elements that have a type attribute with value "text"
$("[attribute!=value]") Selects all elements that either do not have the specific attribute or have the specific attribute whose value is not equal to the specific value. $("[type!='text']") All elements that either do not have a type attribute or have a type attribute whose value is not equal to "text"
$("[attribute|=value]") Selects elements that have the specific attribute with a value either equal to the specific value or starting with the specific value followed by a hyphen (-). $("[title|='Hello']") All elements that have a title attribute whose value is either equal to "Hello" or starting with "Hello" followed by a hyphen
$("[attribute^=value]") Selects elements that have the specific attribute with a value starting with the specific value. $("[title^='Hi']") All elements that have a title attribute whose value is starting with "Hi"
$("[attribute$=value]") Selects elements that have the specific attribute with a value ending with the specific value. $("[title$='bye']") All elements that have a title attribute whose value is ending with "bye"
$("[attribute*=value]") Selects elements that have the specific attribute with a value containing the specific value as a substring. $("[title*='my']") All elements that have a title attribute whose value contains "my" as a substring
$("[attribute~=value]") Selects elements that have the specific attribute with a value containing the specific value as a word. $("[title~='my']") All elements that have a title attribute whose value contains the word "my"
$("[attribute1=value1][attribute2=value2][attributeN=valueN]") Selects all elements that have all of the specific attributes with the specific values $("[type="text"][placeholder=0]") All elements that have a type attribute with value "text" and a placeholder attribute with value 0

Child Selectors

Selector Description Example Selects
$(":first-child") Selects all elements that are the first child of their parent. $("p:first-child") All <p> elements that are the first child of their parent
$(":last-child") Selects all elements that are the last child of their parent. $("p:last-child") All <p> elements that are the last child of their parent
$(":first-of-type") Selects all elements that are the first child of a specific type of their parent. $("p:first-of-type") All <p> elements that are the first <p> element of their parent
$(":last-of-type") Selects all elements that are the last child of a specific type of their parent. $("p:last-of-type") All <p> elements that are the last <p> element of their parent
$(":nth-child(n)") Selects all elements that are the nth-child of their parent (index starts with 1). The parameter can be index, even, odd, or an equation. $("p:nth-child(4)") All <p> elements that are the 4th child of their parent
$(":nth-last-child(n)") Selects all elements that are the nth-child of their parent, counting from the last element(index = 1) to the first. The parameter can be index, even, odd, or an equation. $("p:nth-last-child(1)") All <p> elements that are the last child of their parent
$(":nth-of-type(n)") Selects all elements that are the nth child of a specific type of their parent (index starts with 1). The parameter can be index, even, odd, or an equation. $("p:nth-of-type(4)") All <p> elements that are the 4th <p> element of their parent
$(":nth-last-of-type(n)") Selects all elements that are the nth child of a specific type of their parent, counting from the last element(index = 1) to the first. The parameter can be index, even, odd, or an equation. $("p:nth-last-of-type(1)") All <p> elements that are the last <p> element of their parent
$(":only-child") Selects all elements that are the only child of their parent. $("p:only-child") All <p> elements that are the only child of their parent
$(":only-of-type") Selects all elements that are the only child of a specific type of their parent. $("p:only-of-type") All <p> elements that are the only <p> element of their parent

Content Selectors

Selector Description Example Selects
$(":contains(text)") Selects all elements that contain specific text. $("p:contains(hello)") All <p> elements that contain "hello"
$(":has(selector)") Selects elements which contain the element(s) that matches the specific selector. $("div:has(p)") All <div> elements that have <p> element(s) inside of them
$(":empty") Selects all elements that have no children (including text). $("td:empty") All empty <td> elements
$(":parent)" Selects all elements that are the parent of another element (including text). $("button:parent") All <button> elements that are the parent of another element

Form Selectors

Selector Description Example Selects
$(":button") Selects all button elements and elements of type button. $(".class-name:button") all button elements and elements of type button whose class is "class-name"
$(":checkbox") Selects all elements of type checkbox. $(".class-name:checkbox") All elements of type checkbox whose class is "class-name"
$(":file") Selects all elements of type file. $(".class-name:file") All elements of type file whose class is "class-name"
$(":image") Selects all elements of type image. $(".class-name:image") All elements of type image whose class is "class-name"
$(":input") Selects all <input>, <select>, <textarea>, <buttons> elements. $(".class-name:input") All <input>, <select>, <textarea>, <buttons> elements whose class is "class-name"
$(":password") Selects all elements of type password. $(".class-name:password") All elements of type password whose class is "class-name"
$(":radio") Selects all elements of type radio. $(".class-name:radio") All elements of type radio whose class is "class-name"
$(":reset") Selects all elements of type reset. $(".class-name:reset") All elements of type reset whose class is "class-name"
$(":submit") Selects all elements of type submit. $(".class-name:submit") All elements of type submit whose class is "class-name"
$(":text") Selects all <input> elements of type text. $(".class-name:text") All <input> elements of type text whose class is "class-name"
$(":selected") Selects all options of select elements that are selected. $(".class-name:selected") All options of select elements with class "class-name" that are currently selected
$(":checked) Selects all checkboxes, radio buttons and options of select elements that are checked or selected. $(".class-name:checked") All checkboxes, radio buttons and options of select elements with class "class-name" that are currently checked or selected
$(":disabled) Selects all elements that are disabled. $(".class-name:disabled") All elements with class "class-name" that are disabled
$(":enabled) Selects all elements that are enabled. $(".class-name:enabled") All elements with class "class-name" that are enabled
$(":focus) Selects the element that is currently focused. $(".class-name:focus") The element with class "class-name" that is currently focused

Hierarchy

Selector Description Example Selects
$("ancestor descendant") Selects all descendant elements of a specific ancestor. $("div p") All <p> elements that are descendants of <div> elements
$("parent > child") Selects all child elements specified by "child" of elements specified by "parent". $("div > button") All <button> elements that are children of <div> elements
$("prev + next") Selects all first "next" elements that follow right after the "prev" element. $("p + button") All <button> elements that are the immediately following siblings of each <p> element
$("prev ~ siblings") Selects all "siblings" elements that are following siblings of the "prev" element. $("button ~ p") All <p> elements that are the following siblings of <button> elements

Visibility Selectors

Selector Description Example Selects
$(":hidden") Selects all elements that are hidden. $("button:hidden") All hidden <button> elements and button type <input> elements
$(":visible") Selects all elements that are visible. $("button:visible") All visible <button> elements and button type <input> elements
