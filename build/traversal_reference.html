jQuery Traversing Reference

Ancestors

Method Description Example Returns
.parent([selector]) Gets the parent of the selected element, optionally filtered by a selector. $("p").parent() The parent of <p> elements
.parents([selector]) Gets all ancestors of the selected element, optionally filtered by a selector. $("span").parents() All ancestors of the <span> elements
.parentsUntil([selector] [,filter]) Gets all ancestors of the selected element up to but not including the element that matches the optional selector, optionally filtered by a filter. $("span").parentsUntil("div") All ancestors between <span> and <div> elements (exclusive)
.closest(selector [,context]) Gets the first ancestor of the selected element that matches the selector by traversing up the DOM tree, optionally restricted to the context within which to search for the closest element. $("li#collection").closest("div") Gets the first <div> ancestor of the <li> element which has the id "collection".
.offsetParent() Gets the closest ancestor element that is positioned. An element is positioned if it has a position attribute of relative, absolute, or fixed. $("p").offsetParent() The closest ancestor element of the <p> element that has a position attribute of realtive, absolute, or fixed

Descendants

Method Description Example Returns
.children([selector]) Gets all children of the selected element, optionally filtered by a selector. $("div").children() All child elements of all <div> elements
.find(selector) Gets all descendants of the selected element that match the selector. $("div").find("li") All <li> elements that are descendants of <div> elements

Siblings

Method Description Example Returns
.prev([selector]) Gets the immediately preceding sibling of the selected element, optionally filtered by a selector. $("ul").prev() The immediately preceding siblings of the <ul> elements
.prevAll([selector]) Gets all preceding sibling of the selected element, optionally filtered by a selector. $("ul").prevAll() All preceding siblings of the <ul> elements
.prevUntil([selector][,filter]) Gets all preceding siblings of the selected element up to but not including the element that matches the optional selector, optionally filtered by a filter. $("ul").prevUntil("h2") All preceding siblings between <ul> and <h2> elements (exclusive).
.next([selector]) Gets the immediately following sibling of the selected element, optionally filtered by a selector. If the selector is provided, it will get the immediately following sibling only if it matches the selector. $("h2").next("p") The immediately following sibling of the <h2> elements if it is a <p> element
.nextAll([selector]) Gets all following sibling of the selected element, optionally filtered by a selector. $("h2").nextAll("p") All following siblings of the <h2> elements that are <p> elements
.nextUntil([selector][,filter]) Gets all following siblings of the selected element up to but not including the element that matches the optional selector, optionally filtered by a filter. $("h2").nextUntil("p") All following siblings between <h2> and <p> elements (exclusive).
.siblings([selector]) Gets all the siblings of the selected element, optionally filtered by a selector. $("h3").siblings() All siblings of the <h3> elements

Filtering

Method Description Example Returns
.eq(index) Gets the element with specific index among the selected elements. $("li").eq(1) The second <li> element in the document
.first() Gets the first element of the selected elements. $("li").first() The first <li> element in the document
.last() Gets the last element of the selected elements. $("li").last() The last <li> element in the document
.filter(selector/function(index)) Gets the elements that matches the selector or pass the function’s test. $("p").filter(".test") All <p> elements that have a class "test".
.has(selector) Gets all elements that have the selector inside of them. $("p").has("button") All <p> elements that have button elements inside of them
.is(selector) Checks if the currently selected element matches the selector. $("p#test").parent().is("div") True if the parent of the <p> element with id "test" iis a <div> element; otherwise, false
.not(selector) Gets all elements inside of the selected elements that do not match the selector. $("p").not(".test") All <p> elements that do not have a class "test"
.slice(start [,end]) Gets a subset of currently selected elements starting with a index number. If the end index is specific, the selection ends with the index (exclusive). $("li").slice(0, 2) The first two <li> elements in the document

Iteration

Method Description Example Returns
$(selector).each(function) Executes a function for each element in the selected elements. $("p.class-name").each(function(){this.innerHTML += " executes function"}) Executes the function that adds " executes function" to the end of the text for each <p> element that has a class "class-name"
$(selector).map(callback) Passes each element in the selected elements through a callback function, creating a new jQuery object containing the return values. $("div").map(function(){return this.id;}) A new jQuery object containing each <div> element’s id

Others

Method Description Example Returns
.add(selector [,context]) Adds elements that match the selector to currently selected elements. Context is optional to specify where to start matching the element and the selector. $("p").add("button") The <p> elements and <button> elements
.addBack([selector]) Adds the previous selection to the current selection, optionally filtered by a selector. $("p").next().addBack() The originally selected <p> element and its immediately following sibling.
.contents() Gets the children of selected element, including text and comment nodes. $("p"#found).contents() Contents of the <p> element with id "found"
operations.end() Ends the most recent filtering operation and returns the set of selected elements to its previous state. $("div").find("p").end() All <div> elements
